[English](./README.md) | [简体中文](./README.zh-CN.md)

## 🌟 **Introduction**

`cosmox` is a powerful, all-in-one platform for multimedia management and expansion. It serves as your ultimate personal media library, but also acts as a smart tool for scraping, integrating, and managing diverse resources from the internet. Whether you're building a personal home theater or simply want to centralize video, music, or image files from different sources, this project provides a comprehensive, one-stop solution.
